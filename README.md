This project is some samples of SpringBoot and SpringCloud components<br>

<h3>Components:</h3>

* <b>admin-users:</b> (springboot:web[rest],jpa[postgres],eureka-client,config-client)<br/>

* <b>eureka-server:</b> (springboot:eureka-server)<br/>
