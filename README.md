# Test Spring Cloud

In this project I want to implement a shopping website<br>

<b>Features:</b>

* A default admin user
* Register user
* Add access to users ( admin , readonly )
* Ban/Unban user
* Login
* Logout
* Category management (CRUD)
* Good management (CRUD)
* Order basket (CRUD)
* Order some goods 

<b>Components:</b>

* Security
* Web
* Auto configuration
* Service discovery (Eureka, Ribbon, Feign)
* Load balancing
* API Gateway

<b>Tools:</b>

* Java 19
* Spring Boot 3.0.0
